\section{Descripción del programa 1}\label{umlauts}

Se tiene un alfabeto que sólo posee los caracteres '0' y '1'  y se debe hallar el conjunto de todas las cadenas posibles de longitud que va desde 0 a n, que se pueden obtener combinando estos caracteres. Si se elige n=2, se tendrá $\sum={e,0,1,00,01,10,11}$.
Además, el programa debe tener las opciones manual y automática, en donde n será un número random para ésta última.
